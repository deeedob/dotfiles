# Dotfiles

My [arch](https://archlinux.org/) based dotfiles for the
[Hyprland](https://wiki.hyprland.org) window-manager.

```bash
git clone https://github.com/deeedob/dotfiles.git $HOME/Dotfiles --depth 1
```

# Useful links/repos/etc
